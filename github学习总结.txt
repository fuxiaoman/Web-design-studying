1.注册：
（1）百度GitHub进入官网
（2）点击sign up注册
（3）填写用户名，邮箱，密码――>点击create an account
（4）选择免费的选项――>continue
（5）填写问卷
（6)去所填写的邮箱完成验证。
（7）登陆账号。

2.建立仓库：
（1）左上角加号――>New repository
（2）填写仓库名字（仓库名为英文，试了两次中文都乱码）
（3）基础设置填完――>create repostory
（4）这时候复制仓库的地址

3.安装git，同步本地仓库：
（1）一路默认，我安在C:\Program Files
（2）找到想要保存仓库的目录，右键――>选择git bash here
（3）输入命令 git clone +仓库地址――>回车，同步到本地。
